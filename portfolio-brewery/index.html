Project Details

This mock brewery website was designed and coded in collaboration with one of my classmates at Startup Institute, Jamie Waters. Together we came up with the concept and look for the site, and then coded different features of the site. We created a git repository, worked in separate branches for each feature, pushed our code, ran pull requests, and then merged the final code. We undertook this project as an opportunity to get more familiar with collaborative coding and git.

Project Toolkit: Sublime Text, HTML, CSS, Sass, Zurb Foundation, JavaScript, jQuery, Git, Koala
